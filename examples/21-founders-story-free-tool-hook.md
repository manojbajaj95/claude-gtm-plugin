# The Founder's Story & Free Tool Hook

Write a personal, story-driven LinkedIn post rooted in a real business hurdle, transition into the framework that solved it, and end with a comment-gating CTA that delivers the exact template to anyone who asks.

## Workflow

### Step 1 — Draft the story post

```
Read about/me.md for the founder's voice, background, and past business experiences.
Read strategy/brand.md for the framework or methodology we want to spotlight.

Using core copywriting principles, draft a personal LinkedIn post structured as:

  1. Opening hook (1–2 lines): drop into a specific moment of struggle — a number,
     a date, a conversation, or a feeling. No throat-clearing. Make it scroll-stopping.

  2. The hurdle (3–5 lines): describe the business problem in plain language.
     Be specific about what was at stake. Avoid jargon. The reader should feel it.

  3. The turn (2–3 lines): the moment something shifted — a realization, a decision,
     or a framework that clicked.

  4. The framework reveal (4–6 lines): explain the solution in simple, actionable terms.
     This is the methodology from strategy/brand.md — written as if teaching a friend,
     not pitching a product.

  5. The outcome (2–3 lines): the specific result that followed. Use a real number or
     before/after contrast if available.

Tone: honest, grounded, first-person. No "I'm humbled to share" or LinkedIn-bro clichés.
Target length: 200–250 words before the CTA.
```

### Step 2 — Add the free tool CTA

```
Extend the post from Step 1 with a comment-gating call to action that offers the
exact template or framework used in the story.

CTA structure (append after the outcome):
  - One-line bridge: connect the story outcome to the template
    ("The exact framework I used is now a [one-page template / spreadsheet / checklist].")
  - Instruction: "Comment [WORD] below and I'll send it directly to your DMs."
  - [WORD] should relate to the transformation in the story (e.g., "TEMPLATE", "FRAMEWORK",
    "PLAYBOOK") — not a generic "YES" or "SEND"
  - Optional: add a line that reinforces scarcity or exclusivity without being dishonest
    ("I only check these comments once a day, so if you want it — drop it now.")

The CTA should feel like a natural extension of the story, not a sudden gear shift.
```

### Step 3 — Define the DM and delivery flow

```
Define the follow-through process when someone comments the trigger word.

Deliver:
  - DM copy (under 50 words): warm, personal, delivers the link or attachment
  - Tool or template to deliver: [Insert Template Name or Link]
  - CRM action: tag commenter as free-tool-lead-[tool-slug] and add to nurture sequence
  - Follow-up: 48 hours after DM, send a 1-question reply asking if the template was useful
    (this opens a conversation without being pushy)
```

## Output

- `content/linkedin/drafts/YYYY-MM-DD_founders-story-<topic-slug>.md` — full post including CTA
