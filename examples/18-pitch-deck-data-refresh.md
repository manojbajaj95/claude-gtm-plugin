# The Pitch Deck Data Refresh

Pull your strongest PMF stats, rewrite your value proposition to match current positioning, and build three investor-ready slides that lead with competitive advantage backed by real data.

## Workflow

### Step 1 — Extract the strongest PMF statistics

```
Analyze our latest product-market fit analysis survey results.
Source: [PMF Survey Results File or URL]

Identify:
  - The top 3 user success statistics most compelling to an investor or partner
    (e.g., retention rate, NPS score, "very disappointed" PMF score, time-to-value)
  - Any verbatim user quotes that capture transformation or strong satisfaction
  - The segment of users showing the strongest PMF signal (if segmented data exists)

For each stat provide:
  - The number and what it measures
  - Why it is compelling in an investor context
  - How it compares to industry benchmarks (if known)
```

### Step 2 — Rewrite the core value proposition

```
Read strategy/brand.md for our current brand messaging, positioning, ICP, and
competitive differentiation.
Using the PMF stats from Step 1, rewrite our core value proposition statement.

The new statement should:
  - Lead with the user outcome (not the feature or mechanism)
  - Reference at least one PMF data point to anchor the claim
  - Be usable as the headline of a pitch deck "Why Us" slide
  - Be under 25 words
  - Pass the "so what?" test for a skeptical investor

Also deliver a supporting 2-sentence expansion for use as slide body text.
```

### Step 3 — Design 3 pitch deck slides

```
Using the value proposition from Step 2 and stats from Step 1, produce the content
for 3 new pitch deck slides that communicate our competitive advantage.

Slide 1 — "The Problem We Solve":
  - Headline: frame the market pain in terms of cost or lost opportunity
  - 3-bullet breakdown of the problem (specific, data-backed where possible)
  - Visual suggestion: chart or statistic callout

Slide 2 — "Why We Win" (competitive differentiation):
  - Headline: our key differentiator in one punchy line
  - Comparison table: us vs. top 2 competitors on 4 dimensions that matter to buyers
    (not features — outcomes, speed, cost, simplicity)
  - Visual suggestion: 2×4 comparison grid

Slide 3 — "Proof It Works" (traction/PMF):
  - Headline: lead with the strongest PMF stat from Step 1
  - 3 supporting proof points: mix of quantitative stats and a user quote
  - Visual suggestion: stat callout tiles + pull quote block

For each slide deliver: headline, body content, speaker note (1 sentence),
and visual layout suggestion.
```

## Output

- `content/briefs/drafts/YYYY-MM-DD_pitch-deck-refresh.md` — all 3 slide contents with speaker notes
