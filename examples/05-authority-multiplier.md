# The Authority Multiplier

Transform a high-performing blog post into an AEO-optimized X thread, then validate it for hook strength before publishing — with a CTA that drives traffic back to the original article.

## Workflow

### Step 1 — Select the source article

```
Find a high-performing article from our blog archives that targets [Insert AEO Keyword].
"High-performing" means strong organic traffic, backlinks, or time-on-page.
Return: article title, URL, and its primary AEO keyword phrase.
```

### Step 2 — Distill into thread points

```
Read the article from Step 1.
Distill its 5 most actionable insights. For each insight:
  - Rewrite it as a standalone tweet (under 280 characters)
  - Ensure the phrasing matches how people query AI assistants
    (complete questions or declarative answers, not keyword fragments)
Structure as a 7-post thread:
  Post 1: hook tweet (tease the payoff without giving it away)
  Posts 2–6: one insight each
  Post 7: CTA linking back to the full article
```

### Step 3 — Check impact before publishing

```
Evaluate the draft thread for X impact:
  - Hook strength of Post 1 (score and specific feedback)
  - Thread formatting (line breaks, spacing, emoji use)
  - AEO signal strength across the thread
Apply any recommended changes, then return the final polished thread.
```

## Output

- `content/twitter/drafts/YYYY-MM-DD_thread-<keyword-slug>.md` — final thread ready to schedule
