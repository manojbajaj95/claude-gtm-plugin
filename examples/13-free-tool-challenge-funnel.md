# The Free Tool Challenge Funnel

Identify a tedious recurring task your audience faces, build a simple template or spreadsheet that solves it, and drive sign-ups through a LinkedIn comment-gating sequence that feeds directly into your CRM.

## Workflow

### Step 1 — Identify the recurring task

```
Analyze our target audience's daily workflows based on our ICP in strategy/brand.md.
Identify one specific, recurring task that is:
  - Tedious and time-consuming (not a one-off)
  - Currently solved with manual effort or a clunky workaround
  - Solvable with a spreadsheet, template, checklist, or simple calculator
Return: task description, why it's painful, and the format best suited to solve it.
```

### Step 2 — Design the free tool

```
Using the task identified in Step 1, design the free tool.
Deliver:
  - Tool name (benefit-driven, not feature-driven)
  - Purpose statement (1 sentence: what it does and for whom)
  - Column/section structure (if a spreadsheet or template: list all tabs or sections)
  - 3 sample rows or example outputs that demonstrate its value immediately on open
  - A clear "Done for you" outcome: what the user has at the end of using it
Keep it achievable to build in under 2 hours.
```

### Step 3 — Write the LinkedIn comment-gating posts

```
Read about/me.md for personal voice.
Write a sequence of 3 LinkedIn posts that build toward the tool release:

Post 1 (Tease): Describe the painful task without revealing the solution yet.
  End with: "I built something to fix this. Comment 'TOOL' and I'll send it over."
  Hook: open with a relatable moment or stat. Under 150 words.

Post 2 (Value preview, 3 days later): Share one insight or sample output from the tool.
  Frame it as proof that the tool works. Repeat the comment CTA.
  Under 120 words.

Post 3 (Social proof drop, 3 days after Post 2): Share a reaction or result from
  an early recipient. End with the comment CTA and add urgency (limited sends, etc.).
  Under 100 words.
```

### Step 4 — Set up the CRM capture flow

```
Define the CRM integration flow for the comment-gating campaign:
  Trigger: user comments "TOOL" on any of the 3 posts
  Action sequence:
    1. Send a LinkedIn DM with the tool download link and a 1-line welcome message
    2. Add the contact to CRM with tag: free-tool-[tool-slug]
    3. Enroll in a 3-email nurture sequence (Day 1 delivery, Day 4 follow-up tip,
       Day 10 soft pitch tied to a use case the tool revealed)
Provide the DM copy and Day 1 email copy. Flag which fields to capture in the CRM record.
```

## Output

- `content/linkedin/drafts/YYYY-MM-DD_free-tool-posts-<tool-slug>.md` — 3 LinkedIn posts
- `content/email/drafts/YYYY-MM-DD_free-tool-nurture-<tool-slug>.md` — CRM flow spec + email copy
